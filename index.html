Introduction

uSurvey is a web-based data collection tool designed in partnership with the Uganda Bureau of Statistics (UBOS) to
(i) enhance timely release of survey results,
(ii) increase frequency of collection of information on indicators that would otherwise be updated after long time intervals, (e.g. every 4-5 years); and
(iii) facilitatethe collection of more disaggregated data (i.e. smaller geographical units), thereby enabling the identification and collection of information on marginalized communities.

USSD and ODK Channel

uSurvey enables real-time data collection and storage using mobile device (tablet/phone) through via ODK channels. This secure and interactive connection enables communication through the relay of survey questions from the server to the hand-held mobile device and the instant transmission of each response from the mobile device to the server. The off-line ODK (Open Data Kit) based mobile application to optimize its utilization in locations with intermittent mobile connections.

uSurvey has been developed to collect information from a scientifically selected sample in standard surveys. To this end, it possesses in-built capabilities to randomly select a representative sample from a population of interest and generate a list of unique identification numbers for each sampling unit (e.g. households, institutions, etc.) to be included in the sample. Just like for traditional surveys techniques, uSurvey results can be used to make inferences about the population from which the sample was selected..
