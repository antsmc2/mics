DISCOVER

USSD AND ODK CHANNELS

uSurvey enables real-time data collection and storage using mobile device (tablet/phone) through via USSD and ODK channels. This secure and interactive connection enables communication through the relay of survey questions from the server to the hand-held mobile device and the instant transmission of each response from the mobile device to the server. The off-line ODK (Open Data Kit) based mobile application to optimize its utilization in locations with intermittent mobile connections.

REAL TIME MONITORING

uSurvey eliminates the use of paper questionnaires and associated delays in the release of survey results oftentimes due to the time required for data capture, editing and cleaning. The tool enables remote review and (if necessary) editing of captured data in real time, while the data collectors are still in the field.





Learn more

SAMPLE AND NON SAMPLE SURVEY

uSurvey possesses in-built capabilities to randomly select a representative sample from a population of interest and generate a list of unique identification numbers for each sampling unit (e.g. households, institutions, etc.) to be included in the sample. Just like for traditional surveys techniques, uSurvey results can be used to make inferences about the population from which the sample was selected.
